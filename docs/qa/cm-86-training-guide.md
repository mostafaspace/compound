## 1. Getting Started

### Access the Platform

- **Admin Console:** Navigate to the admin URL (provided by your platform operator)
- **Mobile App:** Download from App Store / Google Play
- **Login:** Use the credentials provided by your compound administrator

### First-Time Setup Checklist

- [ ] Log in with your credentials
- [ ] Verify your profile information
- [ ] Review your compound's building/unit structure
- [ ] Check notification settings
- [ ] Familiarize yourself with the navigation menu

## 2. For Compound Admins

### Daily Operations

| Task | Where | How |
|------|-------|-----|
| View dashboard | Home page | Review compound overview, pending items |
| Check notifications | Bell icon (top right) | Click to view, mark as read |
| Manage visitor entries | Security → Manual Entries | Add/remove visitor records |
| Post announcements | Announcements → New | Write, schedule, publish |

### Managing Buildings & Units

1. Navigate to **Buildings**
2. Click the building name to view floors and units
3. Use **Edit** to modify, **New** to add floors/units
4. For bulk imports, use **Imports** → upload CSV template

### Managing Dues & Payments

1. Navigate to **Dues**
2. View charge types and collection campaigns
3. Start a new campaign to generate charges for all units
4. Track collection rates in **Finance → Reports**

### Inviting Residents

1. Navigate to **Onboarding**
2. Enter resident email/phone
3. Select role (owner/tenant)
4. Send invitation — resident receives email/SMS with activation link

### Work Orders

1. Navigate to **Work Orders**
2. Create → fill title, category, priority, location
3. Submit draft → approve → start → complete
4. Track estimates and reviews

## 3. For Treasurers

### Viewing Financial Reports

1. Navigate to **Finance → Reports**
2. Select report type: revenue summary, collection rate, outstanding balances
3. Filter by date range, building, or unit

### Managing Collection Campaigns

1. Navigate to **Dues**
2. Click **New Campaign**
3. Select charge type, amount, due date
4. Campaign generates charges for all applicable units
5. Monitor collection progress

### Manual Payment Entry

1. Navigate to **Finance → Advanced**
2. Select unit/resident
3. Enter payment amount, method, reference
4. Save — payment recorded, balance updated

---

## 4. For Security Staff

### Managing Visitor Entries

1. Navigate to **Security → Manual Entries**
2. Enter visitor name, purpose, unit visiting
3. Record entry/exit times
4. View history for audit trail

### Gate Management

1. Navigate to **Security → Gates**
2. View gate status (open/closed)
3. Control access remotely if needed

### Reporting Incidents

1. Navigate to **Security → Incidents**
2. Create new incident report
3. Include time, location, description, photos
4. Submit — admin notified automatically

---

## 5. For Residents

### Using the Mobile App

1. Download and install the app
2. Open invitation link or enter compound name
3. Verify identity (document upload if required)
4. Access dashboard: announcements, dues, issues

### Submitting Issues

1. Navigate to **Issues** in the app
2. Select category (maintenance, complaint, suggestion)
3. Add description, photos
4. Submit — admin receives notification

### Viewing Dues

1. Navigate to **Dues** section
2. View outstanding balance, payment history
3. Pay online if payment gateway configured

## 6. Troubleshooting

| Problem | Solution |
|---------|----------|
| Can't log in | Verify email/password; use "forgot password" |
| No notifications received | Check notification settings; ensure app is updated |
| Can't see my compound | Contact admin to verify your compound assignment |
| Payment not showing | Wait 24h for reconciliation; contact treasurer |
| App crashes on launch | Clear cache, reinstall, ensure OS is updated |

2. **Level 2:** Phone support (business hours)
